# Font Dosyaları

Bu klasöre aşağıdaki font dosyalarını eklemeniz gerekmektedir:

## Gotham Font Dosyaları
- `Gotham-Thin.woff2` / `Gotham-Thin.woff` (font-weight: 100)
- `Gotham-XLight.woff2` / `Gotham-XLight.woff` (font-weight: 200)
- `Gotham-Light.woff2` / `Gotham-Light.woff` (font-weight: 300)
- `Gotham-Book.woff2` / `Gotham-Book.woff` (font-weight: 400)
- `Gotham-Medium.woff2` / `Gotham-Medium.woff` (font-weight: 500)
- `Gotham-Bold.woff2` / `Gotham-Bold.woff` (font-weight: 700)
- `Gotham-Black.woff2` / `Gotham-Black.woff` (font-weight: 900)

## Futura PT Font Dosyaları
- `FuturaPT-Light.woff2` / `FuturaPT-Light.woff` (font-weight: 300)
- `FuturaPT-Book.woff2` / `FuturaPT-Book.woff` (font-weight: 400)
- `FuturaPT-Medium.woff2` / `FuturaPT-Medium.woff` (font-weight: 500)
- `FuturaPT-Demi.woff2` / `FuturaPT-Demi.woff` (font-weight: 600)
- `FuturaPT-Bold.woff2` / `FuturaPT-Bold.woff` (font-weight: 700)
- `FuturaPT-Heavy.woff2` / `FuturaPT-Heavy.woff` (font-weight: 800)
- `FuturaPT-ExtraBold.woff2` / `FuturaPT-ExtraBold.woff` (font-weight: 900)

Font dosyalarını bu klasöre ekledikten sonra site otomatik olarak bu fontları kullanacaktır.

**Not:** Eğer font dosyalarınız farklı isimlere sahipse, `app/globals.css` dosyasındaki `@font-face` tanımlamalarını güncellemeniz gerekebilir.
